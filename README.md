## Running the app locally

 1. Get a local copy
  ```
    $ git clone -b master https://github.com/ankit-v2-1/reframe.git
    $ cd reframe
  ```
2. Install dependencies and run locally
  
  ```
    $ yarn install
    $ yarn start

    Navigate to http://localhost:3000
  ```
